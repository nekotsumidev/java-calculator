# 🧮 Java Calculator

A simple calculator made in Java for learning purposes.

This project was created while studying Java and practicing basic programming concepts.

## Features

- Addition
- Subtraction
- Multiplication
- Division

## Technologies

## How to Run

1. Clone the repository

git clone https://github.com/nekotsumidev/java-calculator.git

3. Run the main class
